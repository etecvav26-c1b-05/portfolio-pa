# Portfólio de Programação e Algoritmos

## Sobre o Projeto

Este repositório reúne atividades, exercícios e projetos desenvolvidos na disciplina de Programação e Algoritmos. O objetivo é registrar a evolução dos conhecimentos adquiridos ao longo do curso, demonstrando a aplicação prática de conceitos de lógica de programação, estruturas de dados e desenvolvimento de software.
