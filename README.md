## 📚 Sobre o Repositório

Este repositório reúne atividades, exercícios e projetos desenvolvidos durante a disciplina de **Programação e Algoritmos**.

O objetivo é registrar a evolução dos conhecimentos em lógica de programação, estruturas de decisão, repetição, vetores, funções e resolução de problemas utilizando algoritmos eficientes.
